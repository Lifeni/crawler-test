# 爬虫测试网站

用于 Python 爬虫的测试。

使用 Express.js + EJS + Bootstrap.js + Faker.js 进行编写，网站中涉及的图片来自 Bing 壁纸的 2020 年 4 月、5 月、6 月这三个月的壁纸数据，一共有 90 张。

## 安装和运行

1. 克隆或者下载项目到本地。

2. 进入项目文件夹，在安装 Node 和 npm 的情况下执行：

    ```bash
   $ npm install
   # 或者考虑使用 cnpm：
   # cnpm install 
   ```

3. 运行程序：

    ```bash
    $ npm start
    ```

4. 访问打开 http://localhost:3000，或者使用服务器反向代理 3000 端口。
